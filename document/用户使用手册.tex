\section*{\Large Oops！ I Forget To Return My Book……}

    请您在规定的时限之内归还书籍。\par
    Lisperarian的图书资源是属于每一个爱书的大家的――
    \begin{itemize}
      \setlength{\itemindent}{2em}
      \footnotesize
      \item 不小心误了归还日期？没关系，5日之内不会有进一步的惩罚；
      \item 不过5日之后您的账号将会被暂时封禁，此时您需要与馆长或代馆长联系解决账号问题；
      \item 由此造成的不便，我们深表歉意，更希望您能妥善安排自己的书籍借阅，共建良性的借阅环境。
    \end{itemize}
